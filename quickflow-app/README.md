# QuickFlow App

Application de génération automatique de procédures documentées.

## Installation

```bash
npm install
```

## Lancement

```bash
npm start
```

## Configuration

1. Lancez l'application
2. Cliquez sur ⚙️ (Paramètres)
3. Entrez votre clé API OpenAI (obtenue sur [platform.openai.com](https://platform.openai.com))
4. Cliquez sur "Sauvegarder"

## Utilisation

1. **Sélectionnez** un écran ou une fenêtre à capturer
2. **Capturez** les étapes en cliquant sur "Capturer une étape" à chaque action importante
3. **Terminez** l'enregistrement
4. **Analysez** avec l'IA pour générer les descriptions automatiquement
5. **Générez** le PDF final

## Dossier de sortie

Les PDFs générés sont sauvegardés dans le dossier `/output` de l'application.
